AgroMap

Projeto desenvolvido para a disciplina de Projeto de Sistemas do IFSP Piracicaba.
Aluno: Eraldo Gonçalves Junior
Prof.: Dr. Thiago Luis Lopes Siqueira
Este projeto foi proposto pela empresa @Tech, uma startup que desenvolve tecnologias para o agropecuária, localizada em Piracicaba-SP. Trata-se de um aplicativo desenvolvido com o Framework Xamarin para o registro de ocorrências que serão informadas pelo usuário. Tais eventos devem ter sua geolocalização capturada e registrada pelo dispositivo móvel que for utilizado, que deverá enviar as informações a um servidor.
